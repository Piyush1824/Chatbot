# AI Chat Assistant (Python + Tkinter + Groq)

A desktop AI chatbot with a modern UI, built using **Python Tkinter** and powered by the **Groq LLM API**.

---

## Features

* Multiple chat sessions
* Clean dark-themed interface
* AI responses via LLaMA 3 model on Groq
* Smooth UI using background threading

---

## Requirements

* Python 3.9+
* Groq API Key

Install dependencies:

```bash
pip install groq python-dotenv
```


---

## Setup

Create a `.env` file in the project folder:

```
GROQ_API_KEY=your_api_key_here
```

---

## Run the App

```bash
python your_script_name.py
```
